Usando Thymeleaf com Spring

Vamos criar um controller só pras views.

Esse controller vai renderizar qualquer arquivo HTML que esteja em src/resources/templates

Já na pasta src/resources/static vão ficar imagens, arquivos css e js, etc

Esse controller vai fazer requisições HTTP para a rota da API REST
